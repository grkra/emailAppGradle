This is training project.

------------------------------------------
Compiling and starting the application
------------------------------------------
1. Downoload the code
2. Import it to an IDE
3. Open build.gradle file
4. Start Gradle build script to build the application
5. Start Gradle run script to start the application

------------------------------------------
Building JAR file
------------------------------------------
1. Open buid.grade file
2. Uncomment jar {...} code
3. Run jar code to build portable jar
4. Jar file will be placed in build.libs directory
5. Start the application by doubleclick on jar file
